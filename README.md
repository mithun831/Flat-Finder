# 🏠 FlatFinder Vadodara | Zervis AI

A modern and user-friendly web application to find rental flats in Vadodara with AI-powered assistance.

## 🚀 Features

- 🔍 Filter flats by BHK and budget
- ❤️ Wishlist system (Save properties)
- 📍 Direct Google Maps integration
- 📞 One-click call to property owners
- 🤖 Zervis AI Chatbot assistant
- ⚡ Fast and responsive UI

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage (for wishlist)
- Font Awesome Icons
